PYTHON 爛番茄最爛導演專題-爬蟲簡介

第一階段：爬取維基百科上的"爛番茄0%新鮮度"電影列表

首先，透過自動化程式從維基百科的「爛番茄0%新鮮度電影列表」頁面中擷取數據，並進一步鏈接到爛番茄網站，以抓取每部電影的專屬頁面並保存為.csv檔案。值得一提的是，程序中意外地發現了一部隱藏在維基百科源代碼中的電影《Jeepers Creepers: Reborn》*1，這部2022年作品儘管未列於公開表格中，但實際上符合0%評分的條件，使電影總數達到44部。

第二階段：導演信息的深度挖掘
在此階段，我們透過迴圈結構自動遍歷每部電影的頁面，進而取得每位電影導演的個人資料，包括姓名、生日、出生地及其作品中的最高和最低評分項目。由於部分電影由多位導演合力執導，例如《Transylmania》*2和《The Queen's Corgi》*3，使最終的導演數達到46位。這些資料同樣以.csv格式保存，為後續的分析鋪平道路。

第三階段：全面蒐集導演參與作品數據
最後，我們通過每位導演的專頁，擷取其所有參與的作品，涵蓋導演、編劇、演員等多重身份，並記錄下每部作品的爛番茄專家評分、觀眾評分、上映年份及票房資料。此步驟形成了一個豐富的作品數據集，包含923部作品（749部電影與174部影集），最終因導演合作作品的重複記錄產生975筆觀察數據，其中785筆為電影作品、190筆為影視劇集觀察值。


1 中譯:毛骨悚然：惡魔重生
2 中譯:吸血鬼復仇記
3 中譯:女王的柯基
